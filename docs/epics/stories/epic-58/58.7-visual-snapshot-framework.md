# Story 58.7 — Visual Snapshot Baseline Framework

**Epic:** [Epic 58 — Playwright Test Infrastructure](../../epic-58-playwright-test-infrastructure.md)
**Points:** 5 | **Size:** M
**Status:** Proposed

---

## User Story

**As a** test author,
**I want** a visual snapshot system that captures full-page and element-level screenshots, stores baselines, and compares with configurable threshold,
**So that** visual regressions are caught automatically and baselines are established for every page.

## Description

Build a Playwright visual snapshot system in `tests/playwright/lib/snapshot_helpers.py` that captures full-page and element-level screenshots, stores baselines in `tests/playwright/snapshots/`, compares with configurable pixel-diff threshold, and supports CI auto-update mode.

## Tasks

- [ ] Create `tests/playwright/lib/snapshot_helpers.py`
- [ ] `capture_page_snapshot(page, name)` — full-page screenshot at consistent 1280×720 viewport
- [ ] `capture_element_snapshot(page, selector, name)` — element-level screenshot
- [ ] `compare_snapshot(page, name, threshold=0.001)` — compare against baseline, fail if diff > threshold
- [ ] Configure snapshot storage at `tests/playwright/snapshots/{page_name}/`
- [ ] Platform-aware naming: `{name}-{platform}.png` (linux/darwin/win32)
- [ ] `SNAPSHOT_UPDATE=1` env var for baseline regeneration mode
- [ ] Configurable diff threshold via `SNAPSHOT_THRESHOLD` env var (default 0.1%)
- [ ] `create_baseline(page, name)` — explicitly create/overwrite baseline
- [ ] Integration with pytest: `@pytest.fixture` for snapshot comparison
- [ ] Export from `tests/playwright/lib/__init__.py`
- [ ] Document snapshot workflow in `tests/playwright/SNAPSHOTS.md`
- [ ] Write integration test capturing and comparing Dashboard snapshot

## Acceptance Criteria

- [ ] Full-page snapshots captured at 1280×720 viewport
- [ ] Element-level snapshots crop to exact element bounds
- [ ] Baseline comparison uses pixel-diff with configurable threshold
- [ ] `SNAPSHOT_UPDATE=1` regenerates baselines without failing
- [ ] Platform-aware naming prevents cross-platform false positives
- [ ] Missing baseline auto-creates on first run (does not fail)
- [ ] Diff above threshold produces clear error with actual vs expected paths
- [ ] Snapshot files are .gitignore-friendly (can be committed or CI-generated)

## Definition of Done

- [ ] All snapshot helpers implemented, exported, docstrings complete
- [ ] Dashboard baseline captured successfully
- [ ] Snapshot comparison detects intentional 1px change
- [ ] Documentation written
