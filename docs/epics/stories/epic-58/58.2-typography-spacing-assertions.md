# Story 58.2 — Typography & Spacing Assertion Library

**Epic:** [Epic 58 — Playwright Test Infrastructure](../../epic-58-playwright-test-infrastructure.md)
**Points:** 3 | **Size:** S
**Status:** Proposed

---

## User Story

**As a** test author,
**I want** assertion helpers that validate font family, size, weight, line-height, and spacing against the style guide,
**So that** typography and spacing violations are caught automatically.

## Description

Build helpers in `tests/playwright/lib/typography_assertions.py` that verify font family, font size, font weight, line height, letter spacing, and spacing values against the style guide type scale (Section 6) and spacing system (Section 7).

## Tasks

- [ ] Create `tests/playwright/lib/typography_assertions.py`
- [ ] Implement `assert_typography(page, selector, role)` — validate font-family, font-size, font-weight, line-height for a given role (page_title, section_title, subsection, label, body, caption, kpi, code)
- [ ] Implement `assert_heading_scale(page)` — find all h1–h3 and validate they match Section 6.2 scale
- [ ] Implement `assert_font_family(page, selector, expected)` — check Inter (sans) or JetBrains Mono (mono)
- [ ] Implement `assert_spacing(page, selector, property, expected_px)` — validate padding/margin/gap against 4px grid
- [ ] Implement `assert_density_mode(page, selector, mode)` — validate compact (32px row) / comfortable (40px) / spacious (48px+) per Section 7.2
- [ ] Export from `tests/playwright/lib/__init__.py`
- [ ] Write unit tests

## Acceptance Criteria

- [ ] `assert_typography()` validates all 8 type roles from Section 6.2
- [ ] `assert_heading_scale()` validates h1=20px/600, h2=16px/600, h3=14px/600
- [ ] `assert_spacing()` validates values are multiples of 4px
- [ ] `assert_density_mode()` validates row height for each density mode
- [ ] Font family check normalizes font stack (matches first font in stack)

## Definition of Done

- [ ] All helpers implemented, exported, docstrings complete
- [ ] Unit tests pass
