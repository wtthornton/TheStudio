# Story 58.5 — Interactive Element Test Utilities

**Epic:** [Epic 58 — Playwright Test Infrastructure](../../epic-58-playwright-test-infrastructure.md)
**Points:** 3 | **Size:** S
**Status:** Proposed

---

## User Story

**As a** test author,
**I want** utilities for testing buttons, form submissions, HTMX swaps, tab navigation, dropdowns, and modals,
**So that** page tests can verify all interactive elements work correctly.

## Description

Build utilities in `tests/playwright/lib/interaction_helpers.py` for testing interactive elements. Each utility clicks/interacts and asserts the expected outcome (DOM change, network request, visible feedback).

## Tasks

- [ ] Create `tests/playwright/lib/interaction_helpers.py`
- [ ] `click_and_assert_response(page, selector, expected_text=None, expected_url=None)` — click button, wait for networkidle, assert DOM or navigation change
- [ ] `click_and_assert_htmx_swap(page, selector, target_selector, expected_content)` — click HTMX trigger, wait for target element to update
- [ ] `fill_and_submit_form(page, form_selector, field_map, submit_selector)` — fill fields, submit, return result
- [ ] `switch_tab_and_assert(page, tab_selector, expected_content_selector, expected_text)` — click tab, assert content panel updates
- [ ] `assert_dropdown_behavior(page, trigger_selector, menu_selector)` — open, verify items, close on Escape
- [ ] `assert_modal_behavior(page, trigger_selector, dialog_selector)` — open, verify focus trap, close on Escape
- [ ] `assert_button_feedback(page, selector)` — click, verify visual feedback (loading state, disabled state, or text change)
- [ ] Export from `tests/playwright/lib/__init__.py`
- [ ] Write tests using Settings tabs and Repos registration form

## Acceptance Criteria

- [ ] HTMX swap utility waits for target DOM change, not just network idle
- [ ] Form submission utility returns the result text/status for further assertion
- [ ] Tab navigation utility works with both `hx-get` and regular navigation
- [ ] Modal utility checks focus is trapped and Escape closes
- [ ] All utilities have configurable timeouts

## Definition of Done

- [ ] All utilities implemented, exported, docstrings complete
- [ ] Integration tests pass
