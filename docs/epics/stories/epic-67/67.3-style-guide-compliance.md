# Story 67.3 — Style Guide Compliance

**Epic:** 67 — Compliance Scorecard: Playwright Full-Stack Test Suite
**Points:** 5
**Page:** `/admin/ui/compliance`
**Style Guide:** `docs/design/07-THESTUDIO-UI-UX-STYLE-GUIDE.md`

## User Story

As a platform operator, I want the Compliance Scorecard page to conform to TheStudio UI/UX Style Guide so that the interface is visually consistent with the rest of the admin panel.

## Description

This story validates that the Compliance Scorecard page adheres to the canonical style guide. Tests check typography, color palette, spacing, component patterns, and layout rules. Special attention is given to PASS/FAIL badges (green/red with text), check result indicators, and the repo selector component.

## Tasks

- [ ] Assert page uses the correct font family and sizes per style guide
- [ ] Assert color palette compliance — background, text, accent colors match style guide tokens
- [ ] Assert PASS badge uses green color token and FAIL badge uses red color token per style guide
- [ ] Assert check result pass/fail indicators use consistent styling
- [ ] Assert scorecard card component follows style guide card patterns
- [ ] Assert spacing and padding conform to style guide grid/spacing scale
- [ ] Assert layout structure follows admin page template (sidebar + main content area)
- [ ] Assert repo selector dropdown follows style guide form/select patterns

## Acceptance Criteria

- [ ] Font family, sizes, and weights match style guide definitions
- [ ] Color tokens used for backgrounds, text, and borders match the style guide palette
- [ ] PASS badge renders in green, FAIL badge renders in red, both with text labels
- [ ] Check result indicators follow consistent visual pattern per style guide
- [ ] Scorecard card uses style guide card component pattern
- [ ] Spacing between elements follows the style guide spacing scale
- [ ] Page layout matches the admin shell template structure
- [ ] No ad-hoc styles that contradict the style guide

## Definition of Done

- Playwright test file created and passing
- All style guide compliance checks pass
- PASS/FAIL badge styling assertions verified against style guide tokens
- Test runs green in CI pipeline
