# Story 66.3 — Style Guide Compliance

**Epic:** 66 — Model Gateway: Playwright Full-Stack Test Suite
**Points:** 5
**Page:** `/admin/ui/models`
**Style Guide:** `docs/design/07-THESTUDIO-UI-UX-STYLE-GUIDE.md`

## User Story

As a platform operator, I want the Model Gateway page to conform to TheStudio UI/UX Style Guide so that the interface is visually consistent with the rest of the admin panel.

## Description

This story validates that the Model Gateway page adheres to the canonical style guide. Tests check typography, color palette, spacing, component patterns, and layout rules. Special attention is given to cost values (right-aligned, font-mono), model class badges, and table formatting.

## Tasks

- [ ] Assert page uses the correct font family and sizes per style guide
- [ ] Assert color palette compliance — background, text, accent colors match style guide tokens
- [ ] Assert model class badges use consistent styling (fast, balanced, strong with distinct visual treatment)
- [ ] Assert cost values are right-aligned and use `font-mono` or monospace font family
- [ ] Assert table components use consistent styling per style guide table patterns
- [ ] Assert spacing and padding conform to style guide grid/spacing scale
- [ ] Assert layout structure follows admin page template (sidebar + main content area)
- [ ] Assert spend chart or cost display follows style guide data visualization patterns

## Acceptance Criteria

- [ ] Font family, sizes, and weights match style guide definitions
- [ ] Color tokens used for backgrounds, text, and borders match the style guide palette
- [ ] Model class badges have distinct, consistent visual treatment per style guide
- [ ] Cost and monetary values render right-aligned with monospace font
- [ ] Tables follow style guide table patterns including header styling
- [ ] Spacing between elements follows the style guide spacing scale
- [ ] Page layout matches the admin shell template structure
- [ ] No ad-hoc styles that contradict the style guide

## Definition of Done

- Playwright test file created and passing
- All style guide compliance checks pass
- Cost value formatting and table styling assertions verified
- Test runs green in CI pipeline
