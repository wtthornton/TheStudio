# Story 73.3 — Style Guide Compliance

**Epic:** 73 — Portfolio Health: Playwright Full-Stack Test Suite
**Points:** 5
**Status:** Draft

## User Story

As a platform operator, I want the Portfolio Health page to visually conform to the TheStudio UI/UX Style Guide so that health data is presented consistently with the rest of the admin UI.

## Description

Verify that the `/admin/ui/portfolio-health` page follows the canonical style guide (`docs/design/07-THESTUDIO-UI-UX-STYLE-GUIDE.md`). This includes color palette usage, typography scale, spacing tokens, component recipes (summary cards, tables, status badges, charts), and layout patterns. Health status badges must use the correct color coding.

## Tasks

- [ ] Assert page background uses the correct dark theme color (`bg-gray-950` / `#030712`)
- [ ] Assert card/panel backgrounds use `bg-gray-900` with correct border treatment
- [ ] Assert heading uses the correct typography scale (text-xl or text-2xl, font-semibold, text-white)
- [ ] Assert summary cards follow the style guide card component recipe
- [ ] Assert health status badges use correct color coding (green for healthy, amber for degraded, red for unhealthy)
- [ ] Assert status badges include text labels (not color alone)
- [ ] Assert repo health table uses correct styling (header bg, row borders, text alignment)
- [ ] Assert table headers use `text-gray-400` and `text-xs` or `text-sm` per style guide
- [ ] Assert risk distribution chart/badges use correct color palette
- [ ] Assert spacing between sections follows the spacing token scale (gap-4, gap-6, p-4, p-6)
- [ ] Assert no custom colors or patterns outside the style guide palette
- [ ] Assert responsive layout works at standard breakpoints (1024px, 1280px)

## Acceptance Criteria

- [ ] Dark theme colors match style guide specification
- [ ] Typography scale is consistent across all sections
- [ ] Summary cards, tables, and badges match component recipes
- [ ] Health status badges use correct color coding with text
- [ ] Spacing tokens are consistent
- [ ] No visual regressions from style guide standard

## Definition of Done

- [ ] Playwright style compliance tests written and passing
- [ ] Test committed to `tests/playwright/portfolio-health/`
- [ ] No test flakiness on 3 consecutive runs
- [ ] Story checked off in epic-73 tracker
