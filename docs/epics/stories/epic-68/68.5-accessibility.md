# Story 68.5 — Accessibility WCAG 2.2 AA

**Epic:** 68 — Quarantine: Playwright Full-Stack Test Suite
**Points:** 3
**Status:** Draft

## User Story

As a platform operator using assistive technology, I want the Quarantine page to meet WCAG 2.2 AA standards so that I can navigate, understand, and operate all quarantine features regardless of ability.

## Description

Verify that `/admin/ui/quarantine` passes WCAG 2.2 AA accessibility requirements. Run axe-core automated checks, verify keyboard navigability, confirm proper ARIA labels, validate table `scope` attributes, ensure the destructive Delete button is visually distinct (red variant), and confirm all filter and action controls have associated labels.

## Tasks

- [ ] Run axe-core accessibility scan via Playwright and assert zero violations
- [ ] Assert all `<th>` elements in the event table have `scope="col"` or `scope="row"`
- [ ] Assert the failure reason filter has an associated `<label>` or `aria-label`
- [ ] Assert Replay and Delete buttons have accessible names (visible text or `aria-label`)
- [ ] Assert Delete button uses the destructive variant (visually red, conveys danger)
- [ ] Test keyboard navigation: Tab through filter, table rows, and action buttons
- [ ] Assert focus is visible on all interactive elements (focus ring / outline)
- [ ] Assert color contrast ratios meet AA minimum (4.5:1 for text, 3:1 for large text)
- [ ] Assert status badges convey meaning beyond color alone (text labels present)

## Acceptance Criteria

- [ ] axe-core scan reports zero WCAG 2.2 AA violations
- [ ] Table headers have correct `scope` attributes
- [ ] All form controls and buttons have accessible labels
- [ ] Delete button is identifiable as destructive via color and label
- [ ] Full keyboard navigation works without mouse
- [ ] Focus indicators are visible on all interactive elements
- [ ] Color contrast meets AA thresholds

## Definition of Done

- [ ] Playwright accessibility tests written and passing
- [ ] axe-core scan integrated into test
- [ ] Test committed to `tests/playwright/quarantine/`
- [ ] No test flakiness on 3 consecutive runs
- [ ] Story checked off in epic-68 tracker
