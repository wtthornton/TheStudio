# Story 68.6 — Visual Snapshot Baseline

**Epic:** 68 — Quarantine: Playwright Full-Stack Test Suite
**Points:** 2
**Status:** Draft

## User Story

As a developer, I want visual snapshot baselines for the Quarantine page so that future changes are caught by visual regression tests and unintended UI drift is prevented.

## Description

Capture Playwright visual snapshots of the `/admin/ui/quarantine` page in its key states: populated event list, empty state, filtered view, and post-action states. These snapshots serve as the baseline for visual regression detection in CI.

## Tasks

- [ ] Capture full-page snapshot of the quarantine page with populated event list
- [ ] Capture full-page snapshot of the quarantine page in empty state ("No quarantined events")
- [ ] Capture snapshot of the page after applying a failure reason filter
- [ ] Capture snapshot of a quarantine detail view (single event expanded/selected)
- [ ] Commit all snapshot baselines to the repository
- [ ] Verify snapshots are deterministic (same result on 3 consecutive runs)

## Acceptance Criteria

- [ ] Baseline snapshots captured for: populated list, empty state, filtered view, detail view
- [ ] Snapshots are committed to the repository under the test snapshot directory
- [ ] Snapshot comparison passes on re-run (no false-positive diffs)
- [ ] Snapshots capture the full visible viewport

## Definition of Done

- [ ] Playwright visual snapshot tests written and passing
- [ ] Baseline images committed to `tests/playwright/quarantine/__snapshots__/`
- [ ] No test flakiness on 3 consecutive runs
- [ ] Story checked off in epic-68 tracker
