# Story 68.3 — Style Guide Compliance

**Epic:** 68 — Quarantine: Playwright Full-Stack Test Suite
**Points:** 5
**Status:** Draft

## User Story

As a platform operator, I want the Quarantine page to follow the TheStudio UI/UX Style Guide so that it is visually consistent with the rest of the admin interface and maintains design system integrity.

## Description

Verify that the `/admin/ui/quarantine` page adheres to `docs/design/07-THESTUDIO-UI-UX-STYLE-GUIDE.md`. This includes correct use of color tokens, typography scale, spacing system, component patterns (data table, status badges, filter controls, action buttons, empty state), and layout structure. The destructive Delete button must use the red destructive variant.

## Tasks

- [ ] Assert page uses the correct CSS custom properties / design tokens for colors
- [ ] Assert typography follows the style guide scale (headings, body, captions)
- [ ] Assert spacing between sections and components follows the spacing system
- [ ] Assert data table follows the style guide table pattern (borders, row hover, padding)
- [ ] Assert status badges (pending, replayed, corrected) use correct badge variant styles
- [ ] Assert filter controls follow form control styling from the guide
- [ ] Assert action buttons follow button variant rules (primary, destructive)
- [ ] Assert Delete button uses the destructive (red) variant
- [ ] Assert Replay button uses the appropriate action variant
- [ ] Assert empty state follows the empty state pattern from the guide
- [ ] Assert page layout follows the admin shell grid / layout pattern

## Acceptance Criteria

- [ ] All color tokens match the style guide
- [ ] Typography scale is correct for headings, body, and labels
- [ ] Spacing is consistent with the design system
- [ ] Data table matches the style guide table component
- [ ] Status badges use correct variant styles
- [ ] Delete button is visually destructive (red variant)
- [ ] Empty state follows the documented empty state pattern
- [ ] No invented visual patterns — all patterns from the style guide

## Definition of Done

- [ ] Playwright style compliance tests written and passing
- [ ] Test committed to `tests/playwright/quarantine/`
- [ ] No test flakiness on 3 consecutive runs
- [ ] Story checked off in epic-68 tracker
