# Story 71.3 — Style Guide Compliance

**Epic:** 71 — Settings: Playwright Full-Stack Test Suite
**Points:** 5
**Status:** Draft

## User Story

As a platform operator, I want the Settings page to visually conform to the TheStudio UI/UX Style Guide so that the configuration hub is consistent with the rest of the admin UI.

## Description

Verify that the `/admin/ui/settings` page follows the canonical style guide (`docs/design/07-THESTUDIO-UI-UX-STYLE-GUIDE.md`). This includes color palette usage, typography scale, spacing tokens, component recipes (tabs, forms, buttons, toggles, badges), and layout patterns. The 6-tab navigation and per-tab form layouts must all conform.

## Tasks

- [ ] Assert page background uses the correct dark theme color (`bg-gray-950` / `#030712`)
- [ ] Assert card/panel backgrounds use `bg-gray-900` with correct border treatment
- [ ] Assert heading uses the correct typography scale (text-xl or text-2xl, font-semibold, text-white)
- [ ] Assert tab navigation uses correct active/inactive styling per style guide
- [ ] Assert form labels use correct typography (text-sm, text-gray-400)
- [ ] Assert form inputs use correct styling (bg-gray-800, border-gray-700, text-white)
- [ ] Assert toggle switches use correct on/off colors (bg-blue-600 active, bg-gray-600 inactive)
- [ ] Assert action buttons use correct variants (primary: bg-blue-600, danger: bg-red-600)
- [ ] Assert masked values use `font-mono` styling
- [ ] Assert spacing between sections follows the spacing token scale (gap-4, gap-6, p-4, p-6)
- [ ] Assert status/feedback messages use correct color coding (green for success, red for error)
- [ ] Assert no custom colors or patterns outside the style guide palette
- [ ] Assert responsive layout works at standard breakpoints (1024px, 1280px)

## Acceptance Criteria

- [ ] Dark theme colors match style guide specification
- [ ] Typography scale is consistent across all 6 tabs
- [ ] Tab navigation styling matches style guide tab component recipe
- [ ] Form inputs, buttons, and toggles match component recipes
- [ ] Spacing tokens are consistent
- [ ] No visual regressions from style guide standard

## Definition of Done

- [ ] Playwright style compliance tests written and passing
- [ ] Test committed to `tests/playwright/settings/`
- [ ] No test flakiness on 3 consecutive runs
- [ ] Story checked off in epic-71 tracker
