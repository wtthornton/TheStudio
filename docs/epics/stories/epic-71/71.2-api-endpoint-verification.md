# Story 71.2 — API Endpoint Verification

**Epic:** 71 — Settings: Playwright Full-Stack Test Suite
**Points:** 3
**Status:** Draft

## User Story

As a platform operator, I want all Settings API endpoints to return valid responses so that the settings UI can reliably load and persist configuration data across all 6 tabs.

## Description

Verify that all Settings-related API endpoints return expected HTTP status codes, correct content types, and valid response bodies. This covers the full matrix of GET and POST endpoints for each of the 6 settings tabs, including the reveal and rotation endpoints.
