# Story 63.4 — Metrics: Interactive Elements

**Epic:** [Epic 63 — Metrics Test Suite](../../epic-63-metrics-playwright-suite.md)
**Points:** 3 | **Size:** S
**Status:** Proposed

---

## User Story

**As an** admin operator using the Metrics page,
**I want** all interactive elements to function correctly,
**So that** I can switch time windows and navigate metric categories without encountering broken controls.

## Description

Validate that all interactive elements on `/admin/ui/metrics` function correctly. This includes time window selector changing displayed data (7d, 30d) and possible tab navigation between metric categories (single-pass, loopbacks, reopens).

## Tasks

- [ ] Test time window selector changes displayed data (7d to 30d and back)
- [ ] Test metric values update when time window changes
- [ ] Test tab navigation between metric categories (if present)
- [ ] Test KPI cards are clickable for detail view (if applicable)
- [ ] Test HTMX partial refresh updates metrics without full page reload
- [ ] Test no console errors during interactions
- [ ] Test interactive elements have appropriate cursor styles

## Acceptance Criteria

- [ ] Changing time window selector updates all displayed metric values
- [ ] Time window change does not cause full page reload (HTMX partial)
- [ ] Tab navigation between categories works correctly (if present)
- [ ] No console errors during any interaction
- [ ] All interactive elements show pointer cursor on hover
- [ ] Metric values are consistent after time window toggle

## Definition of Done

- [ ] All tests pass against running Docker stack
- [ ] Tests use shared helpers from Epic 58
