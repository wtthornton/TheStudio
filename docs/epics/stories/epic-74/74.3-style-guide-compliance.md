# Story 74.3 — Style Guide Compliance

**Epic:** 74 — Detail Pages (Repo/Workflow/Expert): Playwright Full-Stack Test Suite
**Points:** 5
**Status:** Draft

## User Story

As a platform operator, I want all detail pages to visually conform to the TheStudio UI/UX Style Guide so that entity details are presented consistently with the rest of the admin UI.

## Description

Verify that the Repo, Workflow, and Expert detail pages follow the canonical style guide (`docs/design/07-THESTUDIO-UI-UX-STYLE-GUIDE.md`). This includes color palette usage, typography scale, spacing tokens, component recipes (detail cards, forms, buttons, badges, tables), and layout patterns across all 3 detail page types.

## Tasks

- [ ] Assert page background uses the correct dark theme color (`bg-gray-950` / `#030712`) on all 3 detail pages
- [ ] Assert card/panel backgrounds use `bg-gray-900` with correct border treatment
- [ ] Assert entity name heading uses the correct typography scale (text-xl or text-2xl, font-semibold, text-white)
- [ ] Assert detail cards follow the style guide card component recipe
- [ ] Assert status badges use correct color coding per status type (tier badges, health badges, pipeline stage badges)
- [ ] Assert action buttons use correct variants (primary: bg-blue-600, danger: bg-red-600, secondary: bg-gray-700)
- [ ] Assert form inputs (tier change, merge mode, profile edit) use correct styling
- [ ] Assert tables (promotion history, workflow steps) use correct styling (header bg, row borders)
- [ ] Assert drift charts/visualizations use correct color palette
- [ ] Assert spacing between sections follows the spacing token scale (gap-4, gap-6, p-4, p-6)
- [ ] Assert labels and metadata use correct typography (text-sm, text-gray-400)
- [ ] Assert no custom colors or patterns outside the style guide palette
- [ ] Assert responsive layout works at standard breakpoints (1024px, 1280px)

## Acceptance Criteria

- [ ] Dark theme colors match style guide specification on all 3 detail pages
- [ ] Typography scale is consistent across all sections
- [ ] Detail cards, buttons, badges, forms, and tables match component recipes
- [ ] Status badges use correct color coding
- [ ] Spacing tokens are consistent
- [ ] No visual regressions from style guide standard

## Definition of Done

- [ ] Playwright style compliance tests written and passing for all 3 detail page types
- [ ] Tests committed to `tests/playwright/detail-pages/`
- [ ] No test flakiness on 3 consecutive runs
- [ ] Story checked off in epic-74 tracker
