# Story 74.4 — Interactive Elements

**Epic:** 74 — Detail Pages (Repo/Workflow/Expert): Playwright Full-Stack Test Suite
**Points:** 3
**Status:** Draft

## User Story

As a platform operator, I want all interactive elements on the detail pages to function correctly so that I can change repo tiers, toggle merge modes, pause/resume repos, rerun verification, escalate workflows, and view promotion history.

## Description

Verify that all interactive elements on the Repo, Workflow, and Expert detail pages respond correctly to user input. This covers tier changes, merge mode toggles, pause/resume actions, rerun/escalate actions, and promotion history loading via HTMX.

## Tasks

### Repo Detail Interactions
- [ ] Write Playwright test for tier change — assert selecting a new tier updates the repo tier display
- [ ] Write Playwright test for merge mode toggle — assert toggling merge mode updates the control state
- [ ] Write Playwright test for pause/resume — assert clicking Pause changes status to paused, Resume changes back
- [ ] Assert promotion history loads via HTMX partial when section is expanded or tab is clicked

### Workflow Detail Interactions
- [ ] Write Playwright test for Rerun Verification button — assert click triggers rerun and shows confirmation
- [ ] Write Playwright test for Escalate button — assert click triggers escalation and shows confirmation
- [ ] Write Playwright test for Send-to-Agent button (if present) — assert click triggers agent assignment
- [ ] Assert pipeline stage visualization updates reflect current state

### Expert Detail Interactions
- [ ] Write Playwright test for drift analysis section — assert drift data loads and displays
- [ ] Assert expert weight/confidence values are displayed and any edit controls function

### Cross-Cutting
- [ ] Assert HTMX partial loads work correctly on all 3 detail pages
- [ ] Assert loading indicators appear during async operations
- [ ] Assert success/error feedback appears after action execution
- [ ] Assert no JavaScript errors in console during interactions

## Acceptance Criteria

- [ ] Repo: tier change, merge mode toggle, pause/resume all function
- [ ] Workflow: rerun verification, escalate, send-to-agent all function
- [ ] Expert: drift analysis loads, weight/confidence display correctly
- [ ] HTMX partial loads work on all detail pages
- [ ] Success/error feedback appears after actions
- [ ] No JavaScript errors during interactions

## Definition of Done

- [ ] Playwright interaction tests written and passing for all 3 detail page types
- [ ] Tests committed to `tests/playwright/detail-pages/`
- [ ] No test flakiness on 3 consecutive runs
- [ ] Story checked off in epic-74 tracker
