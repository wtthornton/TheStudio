# Story 74.2 — API Endpoint Verification

**Epic:** 74 — Detail Pages (Repo/Workflow/Expert): Playwright Full-Stack Test Suite
**Points:** 3
**Status:** Draft

## User Story

As a platform operator, I want all detail page API endpoints to return valid responses so that the Repo, Workflow, and Expert detail UIs can reliably load entity data, perform actions, and display history.

## Description

Verify that all detail page-related API endpoints return expected HTTP status codes, correct content types, and valid response bodies. This covers the full matrix of GET, POST, and PATCH endpoints for repos, workflows, and experts, including partials for merge mode and promotion history.

- [ ] All 16 endpoints return expected status codes
- [ ] GET endpoints return correct content types (JSON or HTML)

- [ ] Partial endpoints return valid HTML fragments
- [ ] No 500 errors under normal conditions
