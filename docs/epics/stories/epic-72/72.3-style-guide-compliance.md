# Story 72.3 — Style Guide Compliance

**Epic:** 72 — Cost Dashboard: Playwright Full-Stack Test Suite
**Points:** 5
**Status:** Draft

## User Story

As a platform operator, I want the Cost Dashboard page to visually conform to the TheStudio UI/UX Style Guide so that cost data is presented consistently with the rest of the admin UI.

## Description

Verify that the `/admin/ui/cost-dashboard` page follows the canonical style guide (`docs/design/07-THESTUDIO-UI-UX-STYLE-GUIDE.md`). This includes color palette usage, typography scale, spacing tokens, component recipes (KPI cards, tables, charts, progress bars), and layout patterns. Currency values must use `font-mono` for alignment.

## Tasks

- [ ] Assert page background uses the correct dark theme color (`bg-gray-950` / `#030712`)
- [ ] Assert card/panel backgrounds use `bg-gray-900` with correct border treatment
- [ ] Assert heading uses the correct typography scale (text-xl or text-2xl, font-semibold, text-white)
- [ ] Assert KPI cards follow the style guide card component recipe
- [ ] Assert currency/cost values use `font-mono` class for tabular alignment
- [ ] Assert tables use correct styling (header bg, row borders, text alignment)
- [ ] Assert progress/utilization bars use correct color scheme (blue for normal, amber/red for warning/critical)
- [ ] Assert provider names and labels use correct typography (text-sm, text-gray-400 for labels)
- [ ] Assert spacing between sections follows the spacing token scale (gap-4, gap-6, p-4, p-6)
- [ ] Assert percentage values and numeric data are right-aligned where appropriate
- [ ] Assert no custom colors or patterns outside the style guide palette
- [ ] Assert responsive layout works at standard breakpoints (1024px, 1280px)

## Acceptance Criteria

- [ ] Dark theme colors match style guide specification
- [ ] Typography scale is consistent across all sections
- [ ] KPI cards, tables, and progress bars match component recipes
- [ ] Currency values use `font-mono`
- [ ] Spacing tokens are consistent
- [ ] No visual regressions from style guide standard

## Definition of Done

- [ ] Playwright style compliance tests written and passing
- [ ] Test committed to `tests/playwright/cost-dashboard/`
- [ ] No test flakiness on 3 consecutive runs
- [ ] Story checked off in epic-72 tracker
