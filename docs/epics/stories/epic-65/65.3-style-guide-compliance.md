# Story 65.3 — Style Guide Compliance

**Epic:** 65 — Tool Hub: Playwright Full-Stack Test Suite
**Points:** 5
**Page:** `/admin/ui/tools`
**Style Guide:** `docs/design/07-THESTUDIO-UI-UX-STYLE-GUIDE.md`

## User Story

As a platform operator, I want the Tool Hub page to conform to TheStudio UI/UX Style Guide so that the interface is visually consistent with the rest of the admin panel.

## Description

This story validates that the Tool Hub page adheres to the canonical style guide. Tests check typography, color palette, spacing, component patterns, and layout rules defined in the style guide. Approval status badges must use the correct color mapping: observe=gray, suggest=blue, execute=purple.

## Tasks

- [ ] Assert page uses the correct font family and sizes per style guide
- [ ] Assert color palette compliance — background, text, accent colors match style guide tokens
- [ ] Assert approval status badge colors: observe=gray, suggest=blue, execute=purple
- [ ] Assert card/list component patterns match style guide specifications
- [ ] Assert spacing and padding conform to style guide grid/spacing scale
- [ ] Assert layout structure follows admin page template (sidebar + main content area)
- [ ] Assert responsive behavior at standard breakpoints (if applicable)
- [ ] Assert tool catalog cards/rows use consistent component patterns

## Acceptance Criteria

- [ ] Font family, sizes, and weights match style guide definitions
- [ ] Color tokens used for backgrounds, text, and borders match the style guide palette
- [ ] Approval badges render with correct colors: observe=gray (#6b7280 or equivalent), suggest=blue (#3b82f6 or equivalent), execute=purple (#8b5cf6 or equivalent)
- [ ] Card or list components follow the style guide component library patterns
- [ ] Spacing between elements follows the style guide spacing scale
- [ ] Page layout matches the admin shell template structure
- [ ] No ad-hoc styles that contradict the style guide

## Definition of Done

- Playwright test file created and passing
- All style guide compliance checks pass
- Badge color assertions verified against style guide tokens
- Test runs green in CI pipeline
